#ETL Sales Pipeline - SQL Server

Proyecto que simula un pipeline ETL completo utilizando SQL Server.

🎯 Objetivo
Demostrar el proceso de extracción, transformación y carga (ETL) de datos para análisis.

🔄 Proceso
Carga de datos desde CSV (staging)
Limpieza y transformación
Modelado para reporting

📊 Resultado
Dataset preparado para análisis de ventas por período.

🛠️ Tecnologías
SQL Server
T-SQL

📂 Estructura
data/: dataset de entrada
scripts/: procesos ETL
docs/: explicación del pipeline
