# ETL Pipeline - Sales

## 📌 Objetivo
Simular un proceso ETL desde datos crudos hasta un modelo listo para análisis.

## 🔄 Flujo ETL

### Extract
Carga de datos desde archivo CSV a tabla staging.

### Transform
- Limpieza de datos
- Filtro de registros inválidos
- Enriquecimiento (year, month)

### Load
Carga a tabla final optimizada para reporting.

## 📊 Resultado
Dataset limpio y estructurado para análisis de ventas.
